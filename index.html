Basado en el artículo de Matt Maribojoc How to Add Drag and Drop to Your VueJS Project.

Notar que el Drag And Drop no funciona en dispositivos móviles. Para corregir esto, recomiendo usar este PolyFill de Bernardo Castillo. Este PolyFill tiene un pequeño error en la línea 152 que no permite que fucnione en notebook/laptop con touch-screen. Corregir como sigue:

Donde dice:

            // listen to touch events
            if ('ontouchstart' in document) {

Debe decir:

            // listen to touch events
            if (navigator.maxTouchPoints) {

La detección de touch-screen usando navigator.maxTouchPoints es más general.

Para probar el Drag And Drop, tome cualquier papeleta y cambiela de bloque o de orden.
